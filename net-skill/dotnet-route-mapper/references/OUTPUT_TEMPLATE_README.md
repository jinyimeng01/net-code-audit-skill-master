## 5. 常见问题

**Q: 为什么某些路由无法解析？**
A: 可能原因：
- 使用了动态路由（如通配符 `{*path}`）
- 使用了自定义中间件/过滤器
- 配置使用了运行时表达式

**Q: 参数类型显示为 `unknown`？**
A: 使用反编译工具 `dnSpy/dnSpy.Console.exe` 获取完整的类定义

**Q: WebForms 路由如何识别？**
A: 检查 .aspx 文件和 web.config 中的 URL 映射配置
