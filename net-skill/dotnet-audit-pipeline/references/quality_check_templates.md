# 质检员校验报告模板

本文件定义质检员（agent-7-x）在每个阶段的**强制填充式校验报告模板**。

质检员完成校验后，必须按照对应模板**逐项填写**所有字段，禁止省略、合并或用一句话概括。每个 `{填写}` 占位符都必须替换为实际内容。

**重要**: 各 skill 已将输出模板外移到 `references/OUTPUT_TEMPLATE*.md` 文件中，采用【填写】占位符的填充式模板。质检时需同时验证输出是否符合对应 skill 的 OUTPUT_TEMPLATE 结构（章节数量、顺序、表格列）。通用输出规范见 `shared/DOTNET_OUTPUT_STANDARD.md`。

## 通用报告结构

每份校验报告必须包含以下三个部分，缺一不可：

````markdown
# 校验报告: {被校验 agent 名称}

## 基本信息
- 质检员: agent-7-{N}
- 校验对象: {被校验 agent 名称}
- 对应 Skill: {加载的 skill 名称}
- 校验文件: {实际读取的输出文件路径，多个用逗号分隔}

## 逐项校验结果

{使用下方对应阶段的校验清单表格，逐行填写}

## 最终判定
- 状态: **通过** / **不通过**
- 通过项: {M}/{N}
- 不通过项清单（如有）:
  - {校验项编号}. {校验项名称}: {具体缺失内容及修复要求}
````
